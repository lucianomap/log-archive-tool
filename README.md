# Log Archiving Tool
BASH script to archive log files.

## Features
- Date Stamp (YYYYMMDD)
- Time Stamp (HHMMSS)
- File saved as `{Directory}_{Date}_{Time}.tar.gz`

## How to Use It
1. Download and change the file permissions.
   
   ``` bash
   chmod +x log-archive-tool.sh
3. Execute the script.
   
   ``` bash
   ./log-archive-tool.sh
